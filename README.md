# Poliklinik App

Poliklinik App adalah aplikasi manajemen poliklinik berbasis web yang dibangun menggunakan Laravel. Aplikasi ini dibuat untuk membantu proses pengelolaan layanan poliklinik, mulai dari pengelolaan data dokter, pasien, jadwal periksa, pendaftaran poli, pemeriksaan pasien, hingga riwayat pemeriksaan.

## Fitur Utama

### Admin
- Mengelola data dokter
- Mengelola data pasien
- Mengelola data poli
- Mengelola data obat
- Melihat data jadwal periksa

### Dokter
- Mengelola jadwal periksa
- Melihat daftar pasien yang mendaftar ke poli
- Melakukan pemeriksaan pasien
- Mengisi catatan pemeriksaan
- Memilih obat yang diberikan kepada pasien
- Melihat riwayat pemeriksaan pasien

### Pasien
- Melakukan pendaftaran ke poli
- Memilih jadwal dokter
- Melihat nomor antrean
- Melihat riwayat pemeriksaan

## Teknologi yang Digunakan

- Laravel 11
- PHP
- MySQL
- Blade Template
- Tailwind CSS
- Composer
- MAMP / XAMPP / Laragon
- Git & GitHub

## Struktur Fitur

Beberapa fitur penting dalam aplikasi ini meliputi:

- Autentikasi pengguna berdasarkan role
- Dashboard sesuai role pengguna
- Manajemen data master
- Pendaftaran pasien ke poli
- Pemeriksaan pasien oleh dokter
- Pengelolaan obat dan resep
- Riwayat pemeriksaan pasien
